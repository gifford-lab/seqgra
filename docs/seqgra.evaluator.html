seqgra.evaluator packageÂ¶

seqgra evaluator hierarchy

Classes:

  • Evaluator: abstract base class for all evaluators

  • FeatureImportanceEvaluator: abstract class for feature importance evaluators

  • MetricsEvaluator: metrics evaluator

  • PredictEvaluator: predict evaluator

  • PREvaluator: PR curve evaluator

  • ROCEvaluator: ROC curve evaluator

  • SISEvaluator: SIS feature importance evaluator
