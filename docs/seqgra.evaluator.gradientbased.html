seqgra evaluator hierarchy - gradient-based feature importance evaluators (FIE)

Classes:

  • GradientBasedEvaluator: abstract class for all gradient-based feature importance evaluators

  • AbstractDifferenceGradientEvaluator: abstract class for difference-based gradient-based feature importance evaluators

  • AbstractGradientEvaluator: abstract class for baseline-based gradient-based feature importance evaluators

  • ContrastiveExcitationBackpropEvaluator: contrastive excitation backprop FIE

  • DeconvEvaluator: deconvolution FIE

  • DeepLiftEvaluator: DeepLIFT FIE

  • DifferenceGradientEvaluator: difference gradient FIE

  • ExcitationBackpropEvaluator: excitation backprop FIE

  • FeedbackEvaluator: feedback FIE

  • GradCamGradientEvaluator: GradCAM FIE

  • DifferenceGradientEvaluator: difference gradient FIE

  • GradientxInputEvaluator: gradient times input FIE

  • GuidedBackpropEvaluator: guided backprop FIE

  • IntegratedGradientEvaluator: Integrated Gradients FIE

  • NonlinearIntegratedGradientEvaluator: nonlinear Integrated Gradients FIE

  • SaliencyEvaluator: absolute gradient (saliency) FIE

  • SmoothGradEvaluator: smooth grad FIE
