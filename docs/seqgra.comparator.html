seqgra.comparator packageÂ¶

seqgra comparators used by seqgras command

Classes:

  • Comparator: abstract base class for all comparators

  • CurveTableComparator: collects PR and ROC curve information in text file

  • FIETableComparator: collects feature importance evaluator information in text file

  • PRComparator: creates PR curves from various grammars and architectures

  • ROCComparator: creates ROC curves from various grammars and architectures

  • TableComparator: collects grammar and model information in text file
